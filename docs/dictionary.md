# Data Dictionary - Pastelería

## Table: cakes (Tabla de Pasteles)
* **cake_id**: Identificador único del pastel (Número).
* **cake_name**: Nombre o sabor del pastel (Texto).
* **price**: Precio de venta al público (Decimal).

## Table: customers (Tabla de Clientes)
* **customer_id**: Identificador único del cliente (Número).
* **first_name**: Nombre del cliente (Texto).
* **last_name**: Apellido del cliente (Texto).
* **phone_number**: Número de teléfono para contacto (Texto).
